## 注意事项与避坑指南

**1. 不要在账户里存太多钱**

随用随充，够用就行。虽然 Pokepay 有合规牌照，但任何集中托管账户都不建议放大额资金。一次充 50-100U 测试 + 消费，用完再充。

**2. IP 比 BIN 更重要**

很多人开不了 ChatGPT Plus，不是卡的问题，是 IP 问题。用机场 IP、数据中心 IP 必拒，要用住宅 IP。地区要和你的 ChatGPT 账号注册地一致。

**3. CVV 错三次会被锁卡**

APP 里输错 CVV / 有效期多次会触发锁定，需要联系官方客服验证本人后解锁。实体卡 PIN 错三次也是同样处理。

**4. 充值网络一定要选对**

USDT 一定选 TRC20，不要选错网络，否则资产可能丢失。充之前先小额测试 5U，确认到账再大额充。

**5. 部分平台不支持**

Steam、Blizzard、EA、Apple Store 内购这些场景 Pokepay 基本搞不定，如果你主要需求是这些，要换其他卡。

**6. 实体卡邮寄有风险**

实体卡从大湾区发出，10-30 个工作日到。偏远地区或海关严的地区可能延迟。如果急用，虚拟卡先顶着。
